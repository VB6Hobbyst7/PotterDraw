Features

  • draw outer wall with Bézier curves (cubic splines); inner wall is generated automatically
  • distort mesh with 3D effects: scallop, ripple, twist, bend, etc.
  • distort texture with parameterized functions: stripes, rings, petals, polar, etc.
  • mesh and texture properties can be modulated with oscillators and animated
  • add color by synthesizing a texture, or use your own image as texture
  • export 3D model as color PLY file, monochrome STL file, or Wavefront OBJ file
  • PLY export supports both texture mapping and vertex color
  • export 2D images as BMP or PNG files
  • record video as sequentially numbered images, convertible to video using ffmpeg or VLC
  • built-in palette editor; print and print preview; full screen mode
  • tabbed MDI interface with dockable bars; unlimited undo and redo
  • library of shapes to use as starting points for your own creations
  • shape tweening
  • context-sensitive help
