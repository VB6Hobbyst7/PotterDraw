*** PotterDraw source distribution ***

REQUIREMENTS:

PotterDraw is intended for Visual Studio 2012. It may compile in later
versions and it can compile in 2010 if you change the Platform Toolset.

INSTALLATION:

The source distribution of PotterDraw doesn't have an installer.

REMOVAL:

To remove PotterDraw, you need to do two things:

1. Delete PotterDraw's files from the folder you unzipped them to.
2. Delete PotterDraw's registry entries, using regedit. The key to
delete is:

"HKEY_CURRENT_USER\Software\Anal Software\PotterDraw"

NOTES:

The binary version of this release is available from the PotterDraw web
site, http://potterdraw.sourceforge.net/.
